# BODR Image Prompt v0.4.0-alpha

**Release theme:** Local ChatGPT OAuth direct image generation and versioned installer

This alpha keeps GitHub Pages as an **Online Read Only Demo** while making the local install much more useful: users can install/update from GitHub Release assets, connect the optional ChatGPT OAuth generation provider, generate images directly from saved prompts, review results in a local inbox, and decide whether to attach an output or save it as a new variant item.

The online demo remains read-only. Add/edit/private library management and direct generation require a local install.

## Highlights

- Added optional local direct generation via **ChatGPT OAuth** using `openai_codex_oauth_native`.
- Added provider-gated generation controls so Generate is only shown when a non-manual provider is configured and authenticated.
- Added `Generate variant` from item detail and a standalone Generate entry for local installs.
- Added GenerationJob result inbox review before writing outputs to the library.
- Added `Attach to current item` for keeping a generated output on the same reference.
- Added `Save as new item` with editable title, collection, tags, prompt, and notes before saving a new variant item.
- Added a compact global generation queue drawer for active, completed, and failed jobs.
- Added friendly policy, rate-limit, auth, and provider failure states.
- Added post-OAuth provider refresh so generation availability updates without a manual page refresh.
- Added versioned release installer/update/rollback:
  - selected release install with `--version`
  - app code under `~/.BODR-Image-Prompt/app/versions/<version>`
  - `app/current` symlink switching
  - private library default outside app code at `~/BODRImagePrompt`
  - built frontend included in the release artifact, so normal installs do not require Node.js
- Updated the GitHub Pages landing page and app banner to **Online Read Only Demo** and added a v0.4 local-generation banner.

## Install

Install the v0.4 alpha from GitHub Release assets:

```bash
curl -fsSL https://raw.githubusercontent.com/EddieTYP/BODR-Image-Prompt/main/scripts/install.sh | bash -s -- --version v0.4.0-alpha
BODR-Image-Prompt start
```

Update an existing install to v0.4:

```bash
BODR-Image-Prompt update --version v0.4.0-alpha
```

Rollback to the previous installed version:

```bash
BODR-Image-Prompt rollback
```

## Known limitations

- This remains an alpha release.
- The Online Read Only Demo is static and read-only; it does not expose Add/Edit/Generate controls.
- ChatGPT OAuth direct generation is local-only and experimental because it depends on the ChatGPT/Codex backend path.
- Fresh OAuth onboarding should still be QA’d carefully on each target environment.
- No interactive release picker/list is implemented yet; use `--version <tag>` for selected releases.
