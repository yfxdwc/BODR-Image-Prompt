# BODR Image Prompt v0.6.3-beta — Sample Metadata Refresh

This beta patch refreshes the public read-only demo and packaged sample manifests after the upstream `awesome-gpt-image-2` gallery changed.

## What changed

- Updated the Online Read Only Demo at <https://eddietyp.github.io/BODR-Image-Prompt/v0.6/> to 533 public references.
- Rebuilt the `awesome-gpt-image-2` sample manifest from the latest upstream snapshot.
- Fixed bilingual prompt parsing so `[中文] ... [English] ...` prompts are split into clean Traditional Chinese, Simplified Chinese, and English variants instead of storing the wrapper text in the Chinese prompt.
- Added demo-only localized display titles for the static GitHub Pages demo. This does not change the local app database schema or backend API.
- Kept public sample sources limited to `freestylefly/awesome-gpt-image-2` and `wuyoscar/gpt_image_2_skill`.

## Updating

```bash
BODR-Image-Prompt update --version v0.6.3-beta
```

After updating the app, re-run the sample import if you want the refreshed local sample library:

```bash
BODR-Image-Prompt sample-data zh_hant awesome-gpt-image-2
```

No database schema change is included. Rollback remains available:

```bash
BODR-Image-Prompt rollback
```
