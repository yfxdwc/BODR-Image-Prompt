# BODR Image Prompt v0.6.0-beta

**Release theme:** Generation Workflow & Attachment Edits

This beta keeps GitHub Pages as an **Online Read Only Demo** while polishing the local generation workflow for day-to-day use: mobile-first composer/stage layout, attachment-based image edits, aspect ratio Auto defaults, result review, and safer Save-as-new metadata handling.

The online demo remains read-only. Add/edit/generation/private library management require a local install.

## Highlights

- Updated the public Pages demo so the current preview lives at `/v0.6/` with a v0.6 landing page, release banner, and archived v0.4/v0.3/v0.2/v0.1 previews.
- Added first-run UI language selection for `繁體中文`, `简体中文`, and `English`, using the existing UI language setting so users can change it later in Config.
- Refined demo copy: online Pages remains read-only, while add/edit/generation and private library management are local-only.
- Improved the mobile generation workflow with a minimal image-first composer/stage layout, compact icon controls, and mobile-friendly history/result review.
- Defaulted new generation jobs to `gpt-5.4`, `High` quality, and aspect ratio `Auto`.
- Added attachment/reference-image support for image edit flows, including using an existing result as the next edit input.
- Preserved prompt purity for aspect ratio Auto by not injecting an aspect-ratio instruction when Auto is selected.
- Kept generation result metadata review, Attach-to-current-item, and Save-as-new item flows focused and compact.
- Changed Save-as-new author defaults so new generated items use the current local user placeholder instead of inheriting the source item author.
- Added v0.7 Account Management planning notes for local accounts, optional passwords, admin/editor/read-only roles, and shared/private migration strategy without adding auth/migration risk to v0.6.

## Install

Install the v0.6 beta from GitHub Release assets:

```bash
curl -fsSL https://raw.githubusercontent.com/EddieTYP/BODR-Image-Prompt/main/scripts/install.sh | bash -s -- --version v0.6.0-beta
BODR-Image-Prompt start
```

Update an existing install to v0.6 beta:

```bash
BODR-Image-Prompt update --version v0.6.0-beta
```

Rollback to the previous installed version:

```bash
BODR-Image-Prompt rollback
```

## Known limitations

- This remains a beta prerelease.
- The Online Read Only Demo is static and read-only; it does not expose Add/Edit/Generate controls.
- ChatGPT OAuth direct generation is local-only and experimental because it depends on the ChatGPT/Codex backend path.
- Running-job cancellation is soft cancellation: the provider request may still finish, but the app ignores/rejects the result if the job has already been cancelled.
- The native generation queue is currently in-process; future releases may add stronger startup recovery and persistence behavior.
- Account Management is planned for v0.7 and is not included in this release.
